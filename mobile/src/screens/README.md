# Screens

This directory contains the main views (screens) of the application.

## Authentication
- **`LoginScreen.js`**: User login form. Handles authentication using `AuthContext`.
- **`RegisterScreen.js`**: User registration form.

## Main App
- **`HomeScreen.js`**: The landing page for authenticated users. Displays a list of Service Stations.
- **`ServicesScreen.js`**: Displays services available at a selected station (including station-specific pricing).
- **`StationsScreen.js`**: (Legacy/Alternate) List of all stations.
- **`BookingScreen.js`**: (Upcoming) Form to book a service at a specific station.
